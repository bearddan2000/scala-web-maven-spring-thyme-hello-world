# scala-web-maven-spring-thyme-hello-world

## Description
A springboot web app with thymeleaf support.

## Tech stack
- scala
- maven
  - springboot
  - thymeleaf

## Docker stack
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
- [Avalble at] (http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
